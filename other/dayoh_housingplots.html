Exploratory plots of Housing data

This data was extracted from http://www.realtor.com using the following search queries for different areas:

  • Centerville, OH
  • Springboro, OH
  • Troy, OH
  • Tipp City, OH

The queries were filtered to houses with 2+ baths, 3+ bedrooms, 2250+ sqft and between 0-20 years. This was further filtered to houses built on or after 2004. However after extracting I saw that there were houses with cities other than the ones uses for queries. Also, this query data was extracted on Oct 21, 2014. So it may not reflect the latest results.
